# converter_dialogflow_intends
Конвертирует JSON файлы dialogflow, которые содержат инфу по интендам, в один csv файл

Разделитель - "табуляция".

Структура csv файла:

Intent    	Phrase
max	        очень довольны
...         ...
Intent    	Phrase
10	        ставлю десять
...         ...
Intent    	Phrase
не цифра	  плохо слышно
...         ...
